# A07 · 信度与效度：从"两个系数"到一套证据体系

> 配套 Q04 的深度版。问卷量表业务线核心长文。

## 先推翻一个流行的简化

多数教科书式的操作是：算个 Cronbach's α > 0.7，做个因子分析载荷 > 0.4，宣布"信效度良好"，翻篇。

这个简化流行到让人忘了它是简化。现代测量学（以《教育与心理测验标准》为代表）的立场是：**效度不是问卷的属性，而是"分数解释与使用"的属性；它不是一个指标，而是一个论证（validity argument）。**你不是在"检验效度"，你是在为"我用这个分数做这个判断是合理的"这个主张收集证据。

这个视角转换不是学究气——它直接改变你做什么、报什么。

## 信度：分数的稳定性

信度回答：同一个人再测一次/换一半题/换个评分者，分数还稳吗？对应三类系数：

**1. 内部一致性**（最常报）：

- **α 的三个使用规范**：分维度算（多维量表全卷混算会虚高或失真）；报告置信区间而不只是点估计；记住 α 是"题目同质+题量"的函数——题目够多的杂烩也能 α>0.8，**α 高不证明单维**；
- **ω 系列**（McDonald's omega）：基于因子模型估计，假设更宽松，`psych::omega()` 即得，正在成为审稿新常态。ω hierarchical 还能回答"总分里有多少方差归属一般因子"——多维量表该不该报总分，看它。

**2. 重测信度**：间隔 2–4 周两次施测的相关。测的是"跨时间稳定性"，只对理论上稳定的构念（人格、能力）有意义，对状态类构念（情绪）重测低是构念本性而非量表缺陷。

**3. 评分者信度**：主观评分场景（作文、开放题）用 ICC 或加权 Kappa；大规模测评中评分员间一致性是必报指标，且要有仲裁机制设计。

**信度的用途标准**：科研中的群体比较 ≥0.7 可用；对个体做高利害判断（选拔、诊断）≥0.9——因为个体决策的误差没有"平均掉"的机会。

## 效度：五类证据

按《标准》的框架，效度证据分五个来源，逐一说实操：

**1. 基于内容的证据**：题目覆盖构念全域了吗？操作：构念蓝图 → 专家逐题评相关性/代表性（量化为 CVI）→ 认知访谈确认目标人群理解无歧义。这是编题阶段的工作，**数据收完就补不了了**。

**2. 基于内部结构的证据**：数据的维度结构符合理论吗？操作：EFA（探索期）→ CFA（验证期，独立样本）→ 报告拟合指标与载荷。进阶：**测量不变性检验**——跨性别/城乡/年龄比较均值之前，要先证明量表在各组中测的是同一个东西、刻度相同（configural → metric → scalar 逐级检验），否则组间差异可能只是测量偏移的假象。IRT 传统下对应的是 DIF 检验，逻辑同源。

**3. 与其他变量关系的证据**：聚合效度（与理论相关的构念确实相关）、区分效度（与理论无关的构念确实区分，AVE 平方根 > 维度间相关）、效标关联（与行为结果/既有金标准的相关，分同时效度与预测效度）。

**4. 基于反应过程的证据**（最被忽视）：被试真的在按你设想的心理过程作答吗？认知访谈、作答时间分析、眼动都算。一道逻辑推理题如果多数人靠猜，分数解释就崩了。

**5. 基于后果的证据**：分数使用带来的实际后果符合预期吗？高利害测评（招生、认证）尤其要审视：对不同群体的通过率差异是否可辩护。

## 把体系装进你的工作流

不同严肃程度的项目，取这套体系的不同子集：

| 场景 | 最低配置 |
| --- | --- |
| 课程作业/初步研究 | 分维度 α + EFA 结构清晰 |
| 学位论文/期刊论文 | ω/α + 分样本 EFA→CFA + 聚合/区分效度 + 内容效度过程描述 |
| 正式测评工具 | 上述全部 + 测量不变性/DIF + 效标研究 + 重测 + 技术手册 |
| 大规模监测 | 再加：评分者信度体系、跨年等值、标准设定效度证据 |

## 三个实战判断

1. **信度是效度的天花板而非通行证**：低信度限制一切相关（衰减公式），但高信度的量表可以稳定地测错东西——"我们的 α 高达 0.93"不回答任何效度问题；
2. **拟合指标不是及格线游戏**：CFI=0.91 不自动合格，CFI=0.89 不自动完蛋。看修正来源、看残差矩阵、看每一步修改有没有理论故事，比卡阈值诚实得多；
3. **效度论证是持续的**：量表换人群（成人量表用于青少年）、换语言、换施测方式（纸笔转在线），既有证据都要重新审视。"这是成熟量表"不是免检章。

把信效度从"过关仪式"升级为"证据体系"，你的方法部分会立刻和 80% 的稿件拉开差距——审稿人看得出谁在走流程、谁在做论证。
