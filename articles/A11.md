# A11 · 潜在回归：背景变量如何正确进入 IRT 模型

> IRT 进阶线。大规模测评方法学的关键一环，也是 PV 的前置知识。

## 问题：先估分数再做回归，错在哪

几乎所有测评项目最终都要回答这类问题：能力与家庭背景、地区、教学变量的关系如何？直觉做法是两步走：先用 IRT 估出每个人的能力 θ̂，再拿 θ̂ 当因变量做回归。

这个"两步法"有一个系统性缺陷：**θ̂ 是带测量误差的点估计，且不同估计方法的误差性质不同，直接进回归会产生方向可预期的偏差。**

- 用 **EAP/MLE 点估计**当因变量：测量误差导致回归系数**衰减**（低估背景变量的效应），且 EAP 的收缩性质会额外扭曲方差与亚群体差异——题目越少，收缩越狠，亚群体间差异被低估越多；
- 更隐蔽的是：EAP 收缩是**向先验均值收缩**。如果估计能力时不带背景信息，所有人向总体均值收缩，弱势群体的真实劣势被"收缩"掉一部分——你的公平性结论直接被方法污染。

矩阵组卷的大规模测评（每人只做部分题）中，个体测量误差巨大，两步法的偏差不可忽略。这就是潜在回归（latent regression）存在的理由。

## 解法：把回归搬进模型里

潜在回归的思想：**不先估个人分数，而是让能力分布本身以背景变量为条件。**模型设定：

$$\theta_n \sim N(\beta_0 + \beta_1 x_{n1} + \dots + \beta_p x_{np},\ \sigma^2)$$

作答数据与背景变量**同时**进入似然，一步估计题目参数、回归系数 β 和残差方差。这样：

1. β 的估计正确传播了测量不确定性，不衰减；
2. 每个人的能力后验以"作答 + 背景"为条件——信息利用最充分；
3. 亚群体比较（性别差异、城乡差异）在模型内直接可得，且无收缩偏差。

PISA、NAEP 等大规模测评的操作流程正是：IRT 测量模型 + 潜在回归（背景变量经主成分压缩后全部放入）→ 从个人后验中抽取**似真值（PV）**供二次分析者使用。PV 本质上就是"潜在回归模型的多重插补输出"——这也解释了 PV 的使用规则为什么和多重插补一模一样（逐个 PV 分析后按 Rubin 规则合并）。

## 实操：R 里怎么做

TAM 包是大规模测评传统的标准工具：

```r
library(TAM)
# Y：背景变量矩阵（回归到能力上）
mod <- TAM::tam.mml(resp = dat, Y = Y_background,
                    irtmodel = "PCM", pweights = w)
mod$beta        # 潜在回归系数
pv <- TAM::tam.pv(mod, nplausible = 10)  # 抽取似真值
```

我们的 IRTC 引擎同样内置潜在回归与个案权重（为大样本场景优化），接口详见 GitHub 仓库文档。

**使用要点：**

1. **背景模型要"宽"**：二次分析可能用到的变量都应进入潜在回归（或其主成分）。某变量不在背景模型里，基于 PV 分析它与能力的关系会向零偏——这是 PV 数据的"隐形合同"：**只有背景模型覆盖的关系才被完整保存**。大规模测评把几百个背景变量压成主成分全放进去，就是为了让这份合同尽量宽；
2. **权重同时进入**：抽样权重在潜在回归估计中就要使用，而不是事后补；
3. **条件化悖论的直觉**：给定总分相同的两个学生，背景变量不同，PV 分布不同——初学者常觉得这"不公平"。要点在于 PV 的设计目标是**群体统计推断**（还原总体的联合分布），不是个体报告。**PV 绝不能用于给个人出分**，这是它的使用边界。

## 什么时候需要这套重武器

| 场景 | 建议 |
| --- | --- |
| 每人做完整测验、题量 40+、只关心个体分数 | 两步法可接受（误差小） |
| 研究"背景 × 能力"关系、题量中等 | 潜在回归或 SEM 一步建模 |
| 矩阵组卷 / 大规模监测 | 潜在回归 + PV，行业标准，没有替代 |
| 使用 PISA 等公开数据 | 按 PV 规则分析（逐 PV + Rubin 合并 + 配套重抽样权重） |

潜在回归是"测量模型"与"结构分析"融合的经典范例。理解了它，你会对"分数"多一层警惕：**任何点估计都是信息压缩，压缩方式本身就带着统计立场。**
