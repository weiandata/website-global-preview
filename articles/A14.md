# A14 · DIF 检测：为"组间差异"结论上保险

> 配套 Q17 的深度版。公平性是所有正式测评的必答题。

## 从一个质疑说起

你的监测报告写着："女生阅读成绩显著高于男生。"评审专家问：**"你怎么知道这是能力差异，而不是题目偏袒？"**

如果题目情境系统性地更贴近某一性别的生活经验（比如大量足球情境的数学题），那么"组间分数差"里就混着"题目偏"的成分。DIF（Differential Item Functioning，题目功能差异）检测就是把这两种东西分离开的质检工序——**没有它，一切组间比较结论都存在未审计的风险**。

## 概念的精确刻画

DIF 的定义带一个关键的条件从句：**在控制能力相同的条件下**，不同组答对某题的概率仍不同。

- 男生某题通过率 65%、女生 55%：这叫 impact，可能只是能力分布不同，**不是 DIF**；
- θ 同为 0.5 的男女生，答对概率 0.7 vs 0.5：这才是 DIF——这道题在能力之外还测了"性别相关的什么东西"。

DIF 分两型：**一致性**（一组在全能力段吃亏）与**非一致性**（低分段 A 组吃亏、高分段反转，ICC 交叉）。检测方法必须能分别捕捉。

## 方法工具箱

**1. Mantel-Haenszel（MH）**：按总分分层，层内比较两组通过率，汇总成共同优势比。
优点：简单、稳健、小样本可用，且有成熟的效应量分级——ETS 的 A/B/C 三级（|ΔMH|<1 为 A 可忽略，>1.5 为 C 需处理），实务沟通极方便。局限：对非一致性 DIF 不敏感。

**2. Logistic 回归法**：建模 P(答对) ~ 匹配分 + 组别 + 匹配分×组别。主效应显著=一致性 DIF，交互显著=非一致性 DIF。配 Nagelkerke ΔR² 做效应量。**一致+非一致通吃、易实现，推荐为默认方法**。

**3. IRT 途径**：

- 似然比检验：比较"该题参数跨组相等 vs 自由估计"两个模型；
- Lord 卡方 / Raju 面积：直接检验两组题目参数差异 / ICC 之间的面积。

理论最干净（匹配变量是潜变量而非含误差的总分），代价是需要较大样本和模型拟合前提。

R 实现：`difR` 包全家桶（MH、logistic、Lord、Raju……一个接口），mirt 的 `DIF()` 与多组模型。

## 实操流程（含最容易做错的一步）

1. **定组**：焦点组/参照组（性别、城乡、语言群体……每对组合单独检测）；
2. **纯化匹配变量（最易做错）**：用"含 DIF 题的总分"当能力代理会污染检测——标准做法是**迭代纯化**：首轮检测 → 把检出题从匹配分中剔除 → 重新检测 → 直至集合稳定；
3. **统计检验 + 效应量双门槛**：大样本下 p 值人人显著，**只有效应量到 C 级（或预定阈值）才进入处理清单**；
4. **内容复审**：统计只能报警，不能定罪。把 C 级题交给内容专家和目标群体代表：这道题为什么偏？是构念本身的合理组成（女生在"阅读文学文本"上的优势可能就是构念的一部分），还是构念无关的偏差（情境经验、方言词汇）？**前者保留，后者修改或删除**——这个判断没有算法能替代；
5. **记录**：检测方法、阈值、检出清单、处置决定，全部入技术文档。

## 三点深水区提醒

1. **DIF 检测不了"整卷都偏"**：所有题都往同一方向偏时，匹配变量本身被污染，DIF 方法集体失灵（这是方法的逻辑边界）。防线只能靠命题阶段的公平性审查和多元命题团队；
2. **多组多题的多重比较**：几百道题 × 多个组对，假阳性要管理（分级效应量本身就是防线，也可加 FDR 校正）；
3. **DIF 是测量不变性的题目级版本**：因子分析传统下的对应物是 scalar invariance 检验，逻辑同源——跨组比较之前，先证明尺子对各组是同一把。

**结论一句话**：组间差异是测评项目最受关注、也最容易被质疑的输出。DIF 检测花的每一天，都是在给报告的公信力买保险。
