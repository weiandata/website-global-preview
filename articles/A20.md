# A20 · 标准设定：分数线背后的方法学

> 配套 Q27 的深度版。测评实务线。

## "及格线"是一个方法学对象

"XX% 的学生达到基本要求"——这句监测报告里最常被引用的话，成立的前提是那条"基本要求"的线划得住脚。划线（标准设定，standard setting）是心理测量里一个成熟的方法学分支，有专门的手册（Cizek & Bunch）、期刊论战和操作规程。本文给出主流方法与完整流程。

## 先立三个认知支柱

1. **分数线本质是价值判断的量化**。"多少算合格"不存在纯统计答案——统计能描述分布，不能告诉你社会应该要求什么。标准设定方法的使命是把这个判断过程**结构化、透明化、可辩护化**；
2. **判断的载体是"边界人物画像"**（borderline examinee）：刚好达标的人是什么样、能做对什么做不对什么。画像立得清，后续判断才有锚；
3. **方法输出的是"建议线"**，最终定线是治理决策（要考虑通过率的社会后果、与往年的衔接）。方法学的作用是让政策考量与专业判断**各归其位、全程留痕**。

## 主流方法详解

**Angoff 法（及其修正版）——以题为中心的经典**
每位专家逐题回答："100 个边界考生中，多少人能答对这道题？"个人的题概率求和 = 个人建议线；专家间经多轮讨论收敛。
优点：逻辑直白、适用各种题型。弱点：让专家凭空估概率认知负担重，估出的概率与真实数据可能系统偏离——所以现代操作必配**数据反馈环节**（告诉专家每题的实际通过率、按当前建议线的整体通过率），在"专家判断"与"经验数据"之间迭代。

**Bookmark 法——以刻度为中心的大规模测评主流**
前提：题目已经 IRT 标定。把全部题目按难度（响应概率 RP67：边界考生有 67% 把握答对的难度点）从易到难装订成"有序题册"，专家翻册放书签："边界考生能稳定掌握到这里。"书签位置的 θ 即建议线。
优点：专家只做"翻到哪页"的判断，认知负担小；天然落在 IRT 刻度上，与等值、多级达标线（A/B/C/D 四级放三个书签）无缝衔接。弱点：依赖标定质量；RP 取值（67% 还是 50%）本身影响结果，须事先定并写明。
国内外大型监测的等级划定，这是事实上的默认方法。

**边界组法 / 对照组法——以人为中心**
请熟悉考生的评价者（教师）**不看分数**把学生分为"明显达标 / 边界 / 明显未达标"，取边界组分数中位数（或达标/未达标两组分布的交点）。
优点：判断对象是活生生的人，不是抽象概率。弱点：依赖评价者对学生的了解质量，且"评价者心中的标准"未必统一。常作为 Angoff/Bookmark 的**交叉验证**手段。

## 严谨流程的完整工序

1. **筹备**：明确政策定义（"基本要求"的文字表述）→ 专家组遴选（10–20 人，学科、地区、性别结构均衡，利益回避）→ 材料准备；
2. **培训与画像校准**：专家亲自做题 → 集体研讨边界人物画像至共识——**这是全流程质量的最大决定因素**；
3. **多轮判断**：第一轮独立判断 → 反馈（专家间分布、实测数据、影响数据即"按此线通过率"）→ 讨论 → 第二轮 → 通常三轮收敛；
4. **汇总与不确定性**：专家判断的均值/中位数为建议线，**同时报告专家间变异（如标准误）**——线是有置信带的，这一点在结果沟通时常被遗忘；
5. **效度证据归档**：程序效度（流程执行记录）、内部效度（专家间一致性、轮间收敛）、外部效度（与其他方法/外部效标的印证）。技术报告的必备章节。

## 三个实战提醒

- **跨年可比先于一切**：等级线要跨年沿用，测验分数必须先等值到同一刻度，否则"达标率提升"可能只是卷子变简单；
- **多条线的一致性**：四等级三条线分别设定时，检查等级宽度是否合理（某等级只占 3 分宽就是信号）；
- **别让影响数据绑架判断**：通过率反馈的目的是让专家知道后果，不是让专家倒推"领导想要的线"。程序上的防线：影响数据在后期轮次才引入，且全程记录判断变化的理由。

一条能被质询的分数线，背后是一份几十页的标准设定技术报告。这个环节的专业投入，买的是整个项目结论的公信力。
