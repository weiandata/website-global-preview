# A05 · CTT 与 IRT 全面对比：什么时候必须换武器

> 配套 Q02 的深度版。适合公众号转载与 B站图解视频。

## 两套理论在争什么

经典测量理论（CTT）统治了测量领域大半个世纪，核心方程朴素到一行：观察分数 = 真分数 + 误差（X = T + E）。项目反应理论（IRT）则把建模对象从"分数"下沉到"人与题的每一次相互作用"。这不是新旧之争，而是**适用场景之争**——本文把两者掰开对比，最后给一张决策表。

## 五个维度的对比

**1. 建模单位：卷子 vs 题目**

CTT 的一切指标都是"整卷"属性：信度是全卷的，标准误是全卷的。换掉三道题，所有指标重算，且与旧版不可比。IRT 把每道题参数化（难度 b、区分度 a），卷子只是题目的集合——于是"换题不换尺"成为可能，这是题库、多卷次、自适应测验的理论基础。

**2. 样本依赖性**

CTT 的题目难度=通过率，本质是"题 × 这批人"的混合量：强的群体让所有题"变简单"。IRT 在模型成立时具有参数不变性：题目参数不随人群漂移，能力估计不随题目组合漂移。**注意这是模型性质，不是魔法**——它成立的前提是模型拟合良好、标定样本足够，实务中要用拟合检验和漂移检测去守护，而不是当成免检的教条。

**3. 测量误差的刻画**

CTT：一个信度系数 + 一个对所有人相同的测量标准误。IRT：信息函数——测量精度随能力水平变化的完整曲线，每个人的能力估计带自己的标准误。实务含义：要在某条分数线上做高利害决策（合格/不合格），IRT 能告诉你**这条线附近**测得准不准，CTT 只能给全卷平均水平。

**4. 对"分数"的定义**

CTT 的分数是有界的总分，天花板/地板处真实差异被压扁，且分数间距无等距保证。IRT 的 θ 定义在无界的潜变量刻度上（logit），有更好的间距性质，可自由线性变换成任何报告量尺（500±100 等）。

**5. 成本**

CTT：几乎零门槛，几十人可算，Excel 能做。IRT：模型假设需要检验（单维性、局部独立、拟合），参数估计需要样本（Rasch 数百，2PL/GPCM 上千），计算与解释都要专业投入。**成本差一个量级，这是 CTT 至今活得很好的根本原因。**

## 常见误解澄清

- **"IRT 比 CTT 准"**——不成立。单次施测、单卷、群体排序的场景，两者的能力排序高度一致（相关常在 0.95+）。IRT 的优势在"可比性、题目管理、精度刻画"这些 CTT 结构上做不到的事，而不是让同一场考试"更准"；
- **"CTT 过时了"**——课堂测验、小样本研究、量表初筛，CTT 是正确工具。用不用 IRT 的判据是需求，不是先进性；
- **"IRT 不需要信度"**——IRT 有自己的信度对应物（边际信度、信息函数），正规报告照样要给。

## 决策表

| 你的场景 | 建议 |
| --- | --- |
| 班级/课堂测验、几十人 | CTT |
| 单次问卷研究、量表分析 | CTT（+因子分析） |
| 量表开发的题目筛选 | CTT 初筛 + IRT 精修（样本够时） |
| 跨年份/跨卷次比较 | **必须 IRT**（等值） |
| 题库建设与维护 | **必须 IRT** |
| 自适应测验 | **必须 IRT** |
| 公平性检验（DIF） | IRT 或基于总分的 MH 法 |
| 大规模素养监测 | IRT 全家桶（标定/等值/PV） |

## 实务中的正确姿势：接力而非替代

成熟项目里两者是流水线关系：CTT 指标（通过率、题总相关、删题后 α）做**第一轮题目体检**，快速、直观、对样本要求低；通过体检的题目进入 IRT 建模做**精细标定**（参数、拟合、DIF）；最终报告在 IRT 刻度上完成等值与转换。CTT 是听诊器，IRT 是核磁共振——医院两个都有，按需使用。

理解这两套理论的边界，是测评数据工作者的基本功。下一篇我会展开"等值"：IRT 参数不变性如何在工程上兑现成"两份卷子一把尺"。
