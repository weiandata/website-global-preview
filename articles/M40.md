# 40 · 难度 b 和区分度 a：怎么判断一道题"好不好"

> 上一篇教你扫参数表，这一篇聚焦一个问题：把 a 和 b 合起来看，怎么判断"这道题到底是好题还是坏题"。给你一套可操作的判断框架。

## 先破一个执念：没有绝对的"好题"

一道题好不好，**不能脱离"这份卷子要干什么"来谈**。同一道题，在一个项目里是宝，在另一个项目里可能是废。

所以判断题好坏，其实是回答两个层面的问题：

1. **这道题本身‘灵不灵’？** —— 主要看区分度 a。
2. **这道题‘用得对不对地方’？** —— 看难度 b 是否契合你的目标人群。

第一个是题的"内功"，第二个是"用人得当"。我们分开说。

## 内功：区分度 a——好坏的硬指标

区分度是**唯一有明确好坏的参数**（第 10 篇）。它衡量题的"分辨力"，越高越好。

一个大致的参照（经验区间，不是铁律）：

| 区分度 a | 大致评价 |
| --- | --- |
| 负值 | 报警！题目出错了，必查（第 46 篇） |
| 接近 0 ~ 0.4 | 太弱，几乎不分辨人，考虑淘汰 |
| 0.4 ~ 0.8 | 一般，可用但不出彩 |
| 0.8 ~ 1.5 | 良好，分辨力不错 |
| 1.5 以上 | 优秀，很能分辨人 |

**记住**：a 是硬指标，太低就是硬伤，跟它测什么、用在哪都没关系。**区分度低的题，先怀疑、后淘汰。**

## 用人得当：难度 b——合适比高低更重要

难度没有好坏，只有**合不合适**（第 9 篇）。判断标准是：**它落在你关心的人群水平范围里吗？**

- 你要测一群中等水平的人，那 b 落在 0 附近的题正合适；b=3 的超难题对他们几乎没用（大家都不会，问不出信息）。
- 你要在及格线（比如 θ=0）上做判断，那你最需要 b 在 0 附近的题（信息山峰正好压在及格线上，第 12 篇）。
- 你要覆盖全人群，那就需要 b 值从低到高**铺满**的一整套题（第 13、15 篇）。

**所以"b=2 的题好不好"这个问题本身没意义**，要问"我这份卷子在 b=2 这个难度段，需不需要它"。需要，它就好；不需要，它再精致也是多余。

## 把 a 和 b 合起来：四象限判断法

把区分度（灵不灵）和难度合适度（用得对不对地方）交叉，一道题落进四个格子之一：

| | 难度合适（在目标段） | 难度不合适（偏离目标段） |
| --- | --- | --- |
| **区分度高** | ⭐ 好题，留着重用 | 好题材，但用错地方——留着，换个项目/换个难度段用 |
| **区分度低** | 位置对但没分辨力——修改或淘汰 | 又没用又不对路——优先淘汰 |

**读法**：

- **左上（高区分 + 难度合适）**：这是梦寐以求的好题，是卷子的顶梁柱。
- **右上（高区分 + 难度不合适）**：题本身是好的，只是当前用不上。别删！存进题库，将来测别的人群时它就派上用场了（这正是题库的价值，第 44 篇）。
- **左下（低区分 + 难度合适）**：位置占对了，可惜不分辨人。先看能不能修改（题干歧义？选项太烂？），改不好就淘汰。
- **右下（低区分 + 难度不合适）**：又废又不对路，第一批出局。

## 别忘了给判断"打折"：标准误和拟合

上面的判断，都建立在"参数估得准"的前提上。所以每次下结论前，补两个动作（第 21、22 篇）：

- **看标准误**：如果 a、b 的标准误很大，那这道题的"好坏"其实没摸准，先别急着定生死，可能是标定不充分。
- **看拟合**：拟合差的题，即便 a、b 数字好看，也说明它"不按套路出牌"，要重点审查是不是测了别的东西或有依赖。

## 一句话收尾

> **判断题好坏分两层：区分度 a 是硬指标（越高越好，低就是硬伤），难度 b 只看合不合适（落在你的目标人群里就好）。用四象限法：高区分+难度合适是宝，高区分+难度不合适存进题库，低区分的修改或淘汰。最后别忘用标准误和拟合给判断打折扣。**
