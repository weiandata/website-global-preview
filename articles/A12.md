# A12 · EAP、MLE、WLE：三种能力估计方法到底差在哪

> IRT 进阶线。选错能力估计方法的后果常被低估，本篇一次讲透。

## 同一个模型，三种"分数"

IRT 模型标定完成后，"给每个人估一个 θ"看似只剩一步，实际上你面临至少三种主流方法：极大似然（MLE）、期望后验（EAP）、加权似然（WLE/Warm 估计）。**同一份作答，三种方法给出的分数不同，性质差异大到足以改变下游结论。**

## 三种方法的机制

**MLE（极大似然估计）**：找让"这个人这套作答"概率最大的 θ。
性质：渐近无偏（题多时）；**全对/全错无解**（似然单调，θ 跑向无穷）——必须打补丁（截断或改用别的方法）；题少时方差大，且在两端有向外的偏差（估计值比真值更极端）。

**EAP（期望后验）**：贝叶斯路线——给 θ 一个先验（通常 N(0,1) 或潜在回归给出的条件先验），取后验分布的均值。
性质：全对全错也有解、数值稳定、均方误差通常最小（这是它成为大规模测评和 CAT 默认的原因）。代价是**收缩（shrinkage）**：估计值被拉向先验均值，题目越少拉得越狠。后果有二：个体层面，极端能力者被低估/高估向中间靠；**群体层面，EAP 分数的方差系统性小于真实能力方差**——用 EAP 分数算方差、相关、组间差异，都会被这个收缩污染（低估）。

**WLE（Warm 加权似然）**：对 MLE 做偏差校正——在似然中乘一个信息函数相关的权重项，抵消 MLE 的一阶偏差。
性质：近似无偏（这是它最大的卖点）、全对全错通常有有限解、无收缩。大规模测评中**报告个体分数时的常用选择**（如 PISA 早年给学校反馈用 WLE），因为"无偏"对个体报告是关键性质。

## 一张速查表

| | MLE | EAP | WLE |
| --- | --- | --- | --- |
| 偏差 | 两端向外偏 | 向先验收缩（向内偏） | 近似无偏 |
| 全对/全错 | 无解 | 有解 | 通常有解 |
| 均方误差 | 较大 | **最小** | 中等 |
| 分数方差 | 偏大 | **偏小（收缩）** | 接近真值 |
| 依赖先验 | 否 | 是 | 否 |
| 典型用途 | 传统/理论分析 | CAT 实时估计、点估计通用 | 个体分数报告 |

## 实务决策：按用途选

**1. 给个人出分、出报告** → WLE 优先（无偏性对个体公平），EAP 次之（但要理解收缩方向并向使用者说明两端的含义）。

**2. CAT 实时估计** → EAP（数值稳、少题时表现好）；终测分数可换 WLE 复核。

**3. 拿分数做二次统计分析（方差、相关、回归、亚群比较）** → **警惕：三种点估计都不理想。**MLE/WLE 带测量误差（衰减效应），EAP 额外带收缩。正确路线按讲究程度递增：

- 最讲究：潜在变量一步建模（SEM/潜在回归，测量误差在模型内处理）；
- 大规模测评标准：似真值 PV + Rubin 合并（见潜在回归篇）；
- 退而求其次：用 WLE + 信度校正衰减，并在局限中说明。

**"拿 EAP 当真分数做相关"是测评二次分析中最常见的系统性错误**，方向可预期：所有关系都被低估。

**4. 排序/选拔场景** → 三者排序高度一致（单维模型下 EAP/WLE 与总分排序也接近），此时方法选择不敏感，选数值稳定的 EAP 即可。

## 两个容易踩的细节

1. **EAP 的先验要和潜在回归一致**：如果标定时用了潜在回归（条件先验），出 EAP 时也应基于同一模型——用错先验，收缩方向都会错；带背景变量的 EAP 给个体出分还有公平性争议（同作答不同背景→不同分数），个体报告场景通常回避（这也是 WLE 的又一优势）；
2. **标准误的口径**：MLE/WLE 的 SE 来自信息函数，EAP 的 SE 是后验标准差，两者含义不同，混报会让"哪些人测不准"的判断失真。技术报告里写清楚用的是哪个。

## 一句话总结

能力估计方法不是软件里随手选的下拉框：**个体报告用 WLE，实时自适应用 EAP，统计推断用 PV/潜变量建模**——把这条原则写进技术方案，能避开一整类"结果解释不动"的事故。
