# 06 · IRT 家族全家福：1PL、2PL、3PL、GRM 都是谁

> IRT 不是一个模型，而是一大家子。这一篇给你介绍几位主要成员，认脸就行，不用记家谱。

## 为什么会有"一家子"模型

前面讲过，题目有几个脾气：难度 b、区分度 a、猜测 c。

IRT 家族的成员，区别就在于**它们各自愿意考虑几个脾气**。有的图省事，只看难度；有的讲究，难度区分度都看；有的连"蒙对"都算进去。考虑的参数越多，模型越贴近现实，但也越"吃"数据、越难伺候。

它们的名字里那个"PL"就是 Parameter Logistic（参数逻辑斯蒂模型）的缩写，前面的数字，就是"考虑几个参数"。所以：

- **1PL = 一个参数**
- **2PL = 两个参数**
- **3PL = 三个参数**

是不是一下就没那么吓人了。逐个见面。

## 老大：1PL / Rasch 模型——只认难度

1PL 只考虑一个参数：**难度 b**。它假设所有题的区分度都一样（大家分辨人的本事相同），也不考虑蒙对。

它有个更有名的名字叫 **Rasch 模型**（以丹麦统计学家 Rasch 命名）。严格说 Rasch 和 1PL 在哲学上有点微妙差别，但对入门的人来说，可以先当成一回事。

**性格**：朴素、稳重、对样本量要求最低（几百人就能跑），结果解释起来最干净漂亮。很多教育测量学派（尤其欧洲）就认它，把"题目区分度都相等"当成一种测量的理想追求。

**软肋**：现实里题目的区分度往往参差不齐，硬套一个"大家都一样"，有些题会拟合不好。

## 老二：2PL——难度 + 区分度

2PL 考虑两个参数：**难度 b 和区分度 a**。它承认"题和题分辨人的本事不一样"，这在心理量表、态度问卷里非常真实。

**性格**：比 1PL 更贴近现实，是社会科学、心理测量里的常客。前面讲能力估计时举的例子（做对难题加更多分）用的就是 2PL 的味道。

**代价**：多估一个参数，就多要一份数据，样本量建议上千。

## 老三：3PL——再加个"蒙对"

3PL 考虑三个参数：难度 b、区分度 a，外加**猜测 c**。

c 专门对付选择题：四选一，完全不会也有 25% 撞对。3PL 承认"哪怕水平极低的人，也有个保底的蒙对概率"，所以它的曲线左端不会贴到零，而是抬起一个小台阶（第 11 篇专门讲这条"抬不起头"的曲线）。

**性格**：最适合有猜测空间的选择题式能力测验（各类标准化考试）。

**代价**：三个参数一起估，最"吃"数据、最不稳定，样本不够时 c 经常估得乱七八糟，需要经验和技巧去驯服。

## 家里的另一支：管"多档打分"的 GRM 等

上面三位管的都是**对错两档**的题（对=1，错=0）。可现实里一大堆题不是非对即错：

- "非常同意 / 同意 / 一般 / 不同意 / 非常不同意"（五档李克特量表）
- 一道大题按步骤给 0、1、2、3 分（部分得分）

这类"多档计分"的题，得请另一支亲戚出马，代表成员有：

- **GRM**（等级反应模型）：适合李克特那种"有序等级"的态度量表，问卷调查里极常见。
- **PCM / GPCM**（部分计分模型）：适合"按步骤给分"的题，比如答对一半给一半分。
- **RSM**（评定量表模型）：适合所有题共用同一套档位（比如全卷都是同一个五点量表）。

你现在不用分清它们的细微差别，只要记住一句：**题目是"对错两档"就用 1PL/2PL/3PL，题目是"多档等级"就用 GRM 那一支。** 想深入可看现有内容库里的 A10 篇。

## 那到底该选谁？——一张速查

| 你的题长这样 | 优先考虑 |
| --- | --- |
| 对错题、想要最简洁稳健、样本不大 | 1PL / Rasch |
| 对错题、题目区分度明显参差、样本上千 | 2PL |
| 选择题、有明显蒙对空间、样本充足 | 3PL |
| 五点/七点李克特态度量表 | GRM |
| 按步骤给分的主观题 | GPCM / PCM |

选模型不是越复杂越好。参数越多越贴近现实，但也越吃数据、越容易翻车。**能用简单模型讲清楚的，就别硬上复杂的**——这是老手的克制。

## 一句话收尾

> **1PL 只看难度，2PL 加区分度，3PL 再加蒙对；李克特这类多档题交给 GRM 一支。名字里的数字，就是"考虑几个参数"。**
