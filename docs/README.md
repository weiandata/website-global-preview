# Documentation

Use this directory for durable project documentation that does not belong in
the root README. Documentation should explain purpose and constraints before
implementation details and should be updated in the same change as the behavior
it describes.

## Contents

- [Website migration guide](WeianData_Website_Repository_Migration_Guide.md)
  defines the approved import and normalization procedure.
- [External dependencies](external-dependencies.md) records browser-loaded
  third-party resources and localization recommendations.
- [Deployment](deployment.md) describes the static-server release model and
  production checks.
- [Repository standard](repository-standard.md) summarizes the baseline applied
  by this repository.
- [Repository Template Development Guide](Repository_Template_Development_Guide.md)
  records the baseline from which this repository was created.

Avoid duplicate policy; link to the Engineering Handbook for normative rules.
